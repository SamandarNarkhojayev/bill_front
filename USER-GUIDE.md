# 🎮 Руководство пользователя

## Быстрый старт

### 1. 🔧 Подготовка Arduino
1. Загрузите прошивку из файла `arduino-firmware.ino` на ESP32-C3
2. Подключите реле к пинам: GPIO 3, 4, 5, 7
3. Подключите кнопки к пинам: GPIO 10, 9, 6, 8 (к GND)
4. Подключите Arduino к компьютеру через USB

### 2. 🖥 Запуск приложения
```bash
npm install
npm run start
```

### 3. 📡 Подключение
1. Перейдите на вкладку **Arduino**
2. Нажмите **🔄** для поиска портов
3. Выберите порт вашего Arduino (например `/dev/cu.usbmodem1201`)
4. Нажмите **Подключить**

### 4. 🎛 Управление
- **Кнопки ВКЛ/ВЫКЛ** - управление через приложение
- **Физические кнопки** - прямое управление на Arduino
- **Лог** - отображает все события в реальном времени

## Интерфейс

### Индикаторы состояния
- 🟢 **Зеленый** - реле включено
- ⚪ **Серый** - реле выключено  
- 🔴 **Красный** - ошибка
- 🟡 **Желтый** - предупреждение

### Сообщения в логе
- `Подключено к /dev/cu.usbmodem1201` - успешное подключение
- `Arduino: ✅ ESP32-C3 4-Relay USB Control Ready` - Arduino готов
- `Реле 1 включено` - изменение через приложение
- `🔘 Кнопка 2 нажата (ВКЛ)` - физическая кнопка

## Решение проблем

### Порт не найден
- Проверьте USB кабель
- Установите драйвера CH340/CP210x
- Попробуйте другой USB порт

### Не подключается
- Закройте Arduino IDE
- Проверьте что порт не занят
- Перезапустите приложение

### Реле не работают
- Проверьте подключение к GPIO 3,4,5,7
- Убедитесь что реле получает питание 5V
- Проверьте настройку ACTIVE_LOW в коде

## Команды Arduino

| Команда | Действие |
|---------|----------|
| `RELAY1_ON` | Включить реле 1 |
| `RELAY1_OFF` | Выключить реле 1 |
| `STATUS` | Получить статус всех реле |

## Технические характеристики

- **Микроконтроллер**: ESP32-C3
- **Количество реле**: 4 (расширяемо до 8+)
- **Интерфейс**: USB Serial, 115200 baud
- **Питание реле**: 5V от ESP32
- **Управление**: Программное + физические кнопки
- **ОС**: Windows, macOS, Linux
