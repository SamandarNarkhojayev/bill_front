## 🎯 Возможности системы

### Управление реле
- **4 независимых реле** (расширяемо до 8+)
- **Программное управление** через графический интерфейс
- **Физическое управление** кнопками на Arduino
- **Real-time синхронизация** состояний

### Интерфейс
- **Современный UI** на React + Tailwind CSS
- **Автоопределение портов** Arduino/ESP32 устройств
- **Цветовые индикаторы** состояния реле
- **Лог событий** в реальном времени
- **Hot reload** для разработки

### Техническое
- **Безопасный протокол** с подтверждениями команд
- **TypeScript** для типобезопасности
- **Electron** для кроссплатформенности
- **SerialPort** для надежной связи
- **Event-driven** архитектура

## 🔧 Настройка Arduino

1. **Загрузить прошивку** `arduino-firmware.ino` на ESP32-C3
2. **Подключить реле** к GPIO 3, 4, 5, 7
3. **Подключить кнопки** к GPIO 10, 9, 6, 8 (на GND)
4. **Подключить USB** к компьютеру

## 🎮 Использование

1. **Запустить** `npm run start`
2. **Выбрать порт** ESP32-C3 в интерфейсе
3. **Нажать "Подключить"**
4. **Управлять реле** через кнопки или физические переключатели
